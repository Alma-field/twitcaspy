**********
Exceptions
**********

Exceptions are available directly in the :mod:`twitcaspy` module, which means
:mod:`twitcaspy.errors` itself does not need to be imported. For example,
:exc:`twitcaspy.errors.TwitcaspyException` is available as
:exc:`twitcaspy.TwitcaspyException`.
